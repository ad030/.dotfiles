# My Dotfiles

My personal config on LMDE 6.

## What I Use

- Wezterm
- Rofi
- Neovim

Using X11:

- i3
  - i3lock-fancy
- Polybar
- Picom

Or using Wayland:

- sway (and swaybar)

## Dependencies

- `Jetbrains Mono` font
- `font-awesome` font
- Neovim 0.10.0+

## Cloning

I store all my config files in the directory `~/dotfiles/`, and create symlinks in `~/.config` to these files.
