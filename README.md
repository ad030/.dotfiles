# My Dotfiles

My personal config on LMDE 6.

## What I Use

- Rofi
- Neovim
- Wayland
  - foot (terminal emulator)
  - grim (screenshot tool)
  - sway (compositor)
  - waybar (bar)

## Dependencies

- `Jetbrains Mono` font
- `font-awesome` font
- Neovim 0.10.0+

## Cloning

I store all my config files in the directory `~/dotfiles/`, and create symlinks to `$HOME` and `$HOME/.config` to these files.
